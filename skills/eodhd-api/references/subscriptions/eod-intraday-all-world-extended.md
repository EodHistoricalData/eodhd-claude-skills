## Notes

- Adds intraday data (1m, 5m, 1h intervals), technical indicators, screener, and WebSocket streaming on top of EOD data
- Does not include company fundamentals, financials, or macro data
- WebSocket streaming covers US Stocks, FOREX, and Cryptocurrencies
- US Ticks API is not included — available only in the All-In-One plans
