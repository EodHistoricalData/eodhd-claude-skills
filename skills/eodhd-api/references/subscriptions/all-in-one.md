## Notes

- This plan bundles all data feeds: EOD, intraday, fundamentals, calendars, technicals, screener, and WebSocket streaming
- The only feature not included is Extended Fundamentals — see [All-In-One with Extended Fundamentals](all-in-one-extended-fundamentals.md)
- US Ticks API is included (not available on lower-tier plans)
- Best value for users who need access to multiple data types
