## Notes

- This plan is focused on company fundamentals, financial statements, and macro data
- Does not include price data (EOD, intraday, live quotes) — pair with an EOD plan if you need both
- Includes Calendar API (earnings, IPOs, splits, dividends), insider transactions, and macro indicators
- Stock market logos (40,000) are included
- Delisted tickers data is accessible for historical research
