## Notes

- This plan is focused on financial event calendars — earnings, IPOs, splits, and dividends
- Includes Earnings Per Share data alongside the calendar endpoints
- Does not include price data, fundamentals, intraday, or WebSocket streaming
- Good for building earnings calendars, event trackers, or dividend schedule tools
- Pair with an EOD or Intraday plan if you also need price data
